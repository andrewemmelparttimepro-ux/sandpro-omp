Executive Summary

The Organization chart now uses a hybrid Bamboo-style layout: top-level leadership stays horizontal, while deeper reporting groups collapse into compact vertical rails. The chart is easier to scan, uses a smaller 2000 x 1200 working canvas, and keeps Fit/Root/zoom controls in one dedicated canvas control cluster.
